单个限制的编辑接口
----------

接口地址
----------
  * 正式接口：/userreview/function/restrict-edit

请求参数说明
----------
|  参数         |说明          |是否必选|
| ------------- |:-------------|:-----:|
| id      | 限制id |必选    |
| ids      | 选项的id集合，如1,2,3 |必选    |

返回示例
----------
<pre>
{
    "err": 0,
    "msg": "success",
    "data": []
}
</pre>

返回结果说明
----------
<pre>
2.107	 编辑菜单权限限制项
</pre>
