修改用户渠道类型和所属渠道商接口
----------

接口地址
----------
  * 正式接口：/userreview/canal/user-edit

请求参数说明
----------
|  参数         |说明          |是否必选|
| ------------- |:-------------|:-----:|
| username            | 用户名 |可选    |
| canaltype           | 渠道分类，若是终端用户则传空 | 可选    |
| usercanaltype       | 用户类型，0终端,1渠道,2渠道客 |可选    |
| canalusername       | 所属渠道商用户名 |可选    |

----------

    msg: "success",
    err: 0
}
